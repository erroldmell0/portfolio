# Personal Portfolio Website

A responsive personal portfolio website built using **React** and **Tailwind CSS** to showcase my projects, skills, and experience. The website features a modern UI, smooth scroll animations, and is resposive.

---

## Features

- Responsive navigation bar with mobile menu
- Fixed navbar for easy access
- Scroll-based reveal animations
- Clean and minimal UI
- Downloadable resume (CV)
- Fully responsive across devices

---

## Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Animations:** CSS Transitions & Intersection Observer
- **Deployment:** Vercel
